Installation
============

Install my-project with npm

.. code:: bash

     npm install my-project
     cd my-project

Run Locally
===========

Clone the project

.. code:: bash

     git clone https://link-to-project

Go to the project directory

.. code:: bash

     cd my-project

Install dependencies

.. code:: bash

     npm install

Start the server

.. code:: bash

     npm run start
